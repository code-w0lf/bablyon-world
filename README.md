# Babylon-world

My first Babylon.JS project. First person shooter game, trying to show my knowledge of the Babylon.JS library and to have a bit of fun.

## Installation

```
npm install
```

## Build

```
npm build
```

## Running localally (port 8080)

```
npm start
```

## Gameplay

- WASD - Movement
- Shift - Hold to move faster
- 1 - Thrid Person View
- 2 - First Person View
